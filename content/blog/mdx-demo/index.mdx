# MDX 功能展示

这篇文章将展示 MDX 支持的各种功能和用法。MDX 允许我们在 Markdown 中使用 React 组件，这让文章写作变得更加灵活和强大。

## 基础 Markdown 语法

### 文本格式化

你可以使用 **粗体**、*斜体* 或 ***粗斜体*** 来强调文本。

### 列表

无序列表：

- 第一项
- 第二项
  - 子项 1
  - 子项 2
- 第三项

有序列表：

1. 第一步
2. 第二步
3. 第三步

### 引用

> 这是一段引用文本。
> 
> 引用可以包含多个段落。

### 代码

行内代码：`console.log('Hello, World!')`

代码块：

```javascript
// 这是一个 JavaScript 函数
function greeting(name) {
  return `Hello, ${name}!`;
}

console.log(greeting('MDX'));
```
